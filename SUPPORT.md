# Support

This library is maintained for production use in supported Spring Boot
applications.

Support focuses on current releases, compatibility with supported Spring Boot
versions, clear documentation, and reliable behavior for scheduled job audit
events.

Use GitHub Issues for bug reports, compatibility problems, and usage questions.

## Supported Versions

Support is provided for the latest released version of this library.

Fixes for older versions are evaluated case by case based on severity, impact,
compatibility risk, and maintenance effort.

## Getting Help

Use GitHub Issues for:

- Bug reports
- Usage questions
- Documentation issues
- Feature requests

Before opening an issue, please check the README, changelog, and existing issues
to see whether the question has already been answered.

## Bug Reports

When reporting a bug, please include:

- Library version
- Spring Boot version
- Java version
- Relevant configuration
- Expected behavior
- Actual behavior
- A minimal reproduction, if possible

## Security Issues

Do not report security vulnerabilities through public GitHub issues.

Follow the instructions in [SECURITY.md](SECURITY.md) for private vulnerability
reporting.

## Support Scope

Public support is provided through GitHub Issues on a best-effort basis.

Organizations that need guaranteed response times, production support, or
private implementation guidance should arrange those expectations separately
with the maintainer.
