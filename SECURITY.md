# Security Policy

## Supported Versions

Security fixes are provided for the latest released version of this library.

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Use GitHub Private Vulnerability Reporting to report suspected security issues privately to the maintainer.

Please include, if possible:

- a description of the issue
- affected version(s)
- steps to reproduce
- impact assessment
- any suggested remediation

Valid reports will be reviewed and addressed in a future release as appropriate.
